## Variants

Actionable Alert is available in many colour variants. To include a 'ghost' button, include the corresponding button HTML inside the `ActionableAlert` element and define `primaryButtonText`.

## Binary

Each variant above can render a second 'outline' button. To include this button, include the corresponding button HTML inside the `ActionableAlert` element and define `secondaryButtonText`.
