### Selected

Icon buttons sometimes need a ‘selected’ state. For example, an icon button is sometimes used to open up a small menu of options. Clicking once drops down a menu, and the button should enter a ‘selected’ state for visual clarity on what button the menu belongs to. Clicking the button again closes the menu and the button would revert to the ‘not selected’ state.
